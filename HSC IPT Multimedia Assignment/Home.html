Software Development Approaches

Are you ready to learn?

When developing new software to be implemented into a system to perform processes that no other available software packages can offer the method used to develop and implement such a software package must be considered prior to starting. The user of the software; those who require new software and will use it afterwards, must consider factors such as:

- Budget

- Who will use it

- Size of the project

- (Type of) Information collected

- Time

- Requirements of the new software

 

The method undertaken to create new software is known as the approach. There are 5 different approaches which vary in the way they aim to design and create new software packages. They can range from being very simple and basic to being very complex, large, formal and well-structured. These 5 approaches are known as:

- Structured Approach

- Agile Approach

- Prototyping Approach

- RAD Approach

- End-User Approach
